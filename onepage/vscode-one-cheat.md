VS CODE ONE PAGE CHEAT
===

## COMMAND PALETTE & COMMANDS

- `Ctrl+Shift+P` / `F1` : Open Command Palette  
- `Ctrl+K Ctrl+S` : Keyboard Shortcuts editor  
- `Ctrl+P` : Quick Open (fuzzy find files)  
- `Ctrl+Shift+N` : New VSCode window  
- `Ctrl+Shift+W` : Close VSCode window  

---

## NAVIGATION & SYMBOLS

- `Ctrl+P` : Go to file by name  
- `Ctrl+G` : Go to line  
- `Ctrl+Shift+O` : Go to symbol (functions/vars)  
- `Ctrl+T` : Show all symbols  
- `Alt+Left` / `Alt+Right` : Navigate back / forward  
- `Ctrl+Shift+Tab` : Editor history (reverse)  

---

## SEARCH & MULTI-CURSOR

- `Ctrl+F` : Find in current file  
- `Ctrl+H` : Replace in current file  
- `Ctrl+Shift+F` : Find across workspace  
- `Ctrl+D` : Add next match to multi-cursor  
- `Ctrl+Shift+L` : Select all occurrences  
- `Ctrl+Alt+Up/Down` : Add cursor above / below  
- `Alt+Click` : Insert cursor at click position  

---

## EDITING & TEXT MANIPULATION

- `Ctrl+X` / `Ctrl+C` : Cut / Copy line if no selection  
- `Shift+Alt+Down/Up` : Copy line down / up  
- `Alt+Up/Down` : Move line up / down  
- `Ctrl+Shift+K` : Delete line  
- `Ctrl+Enter` : Insert line below  
- `Ctrl+Shift+Enter` : Insert line above  
- `Ctrl+Shift+\` : Jump to matching bracket  
- `Ctrl+]/[` : Indent / Outdent  

---

## FOLDING & STRUCTURE

- `Ctrl+Shift+[` : Fold region  
- `Ctrl+Shift+]` : Unfold region  
- `Ctrl+K Ctrl+[` : Fold all subregions  
- `Ctrl+K Ctrl+]` : Unfold all subregions  
- `Ctrl+K Ctrl+0` : Fold all regions  
- `Ctrl+K Ctrl+J` : Unfold all regions  

---

## TAB, PANEL & VIEW MANAGEMENT

- `Ctrl+B` : Toggle sidebar  
- `` Ctrl+` `` : Toggle integrated terminal  
- `Ctrl+Shift+` `` : Create new terminal  
- `Ctrl+Shift+E` : Focus Explorer  
- `Ctrl+Shift+F` : Focus Search  
- `Ctrl+Shift+G` : Focus Source Control  
- `Ctrl+Shift+D` : Focus Debug  
- `Ctrl+Shift+X` : Focus Extensions  
- `Alt+1 / Alt+2 / ...` : Focus sidebar group by index  

---

## CODE NAVIGATION

- `F12` : Go to Definition  
- `Alt+F12` : Peek Definition inline  
- `Ctrl+K F12` : Open Definition to side  
- `Ctrl+.]` : Go to next symbol  
- `Ctrl+Shift+M` : Show Problems panel  
- `F8` / `Shift+F8` : Next / Previous error or warning  

---

## REFACTORING & QUICK FIXES

- `Ctrl+.` : Quick Fix / Code Action  
- `F2` : Rename symbol  
- `Shift+Alt+F` : Format document  
- `Ctrl+K Ctrl+F` : Format selection  

---

## ADVANCED SEARCH & REPLACE

- `Ctrl+Shift+H` : Replace in files  
- `Alt+Enter` : Select all find matches  
- `Ctrl+K Ctrl+D` : Move last selection to next match  
- Search modifiers: `Alt+C` / `Alt+R` / `Alt+W` : Toggle case / regex / whole word  

---

## POWER USER & WORKFLOWS

- Quick open any command: `Ctrl+Shift+P` → type name  
- Multi-cursor edits: `Ctrl+D` + typing  
- Search & replace all instances: `Ctrl+Shift+F` → `Alt+Enter` → type replace  
- Terminal toggle: `` Ctrl+` ``  
- Split terminal: `Ctrl+Shift+5` (custom binding)  
- Pin/unpin panels for context switching
